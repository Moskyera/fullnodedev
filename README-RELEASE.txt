HAC Miner — FULL PACKAGE (Windows x64)

FOR: Clean PC — everything in one ZIP

QUICK START
-----------
1. Extract ZIP to a folder (e.g. C:\HacashMiner)
2. Run SETUP.bat
3. Open miner-panel.exe → Settings → wallet → Start
